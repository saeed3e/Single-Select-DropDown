Options

.........................

active Type: Integer

Default: 0
Which panel is currently open.
  • Boolean: Setting active to false will active zero-based index of the panel and true will active first index of the panel
  • Integer: The zero-based index of the panel that is active (open). A negative value selects panels going backward from the last panel.
Code examples:

Initialize the Accordion with the active option specified:

1
 
$( ".selector" ).accordion({ active: 1 });

disabledType: Boolean

Default: false
Disables the accordion if set to true.
Code examples:

Initialize the accordion with the disabled option specified:

1
 
$( ".selector" ).accordion({ disabled: true });

openEventType: String

Default: "click"
The type of event that the accordions should react to in order to activate the accordion. To activate on hover, use "mouseover".
Code examples:

Initialize the accordion with the openEvent option specified:

1
 
$( ".selector" ).accordion({ openEvent: "mouseover" });

collapsibleType: Boolean

Default: false
Whether all the sections can be closed at once. Allows collapsing the active section.
Code examples:

Initialize the accordion with the collapsible option specified:

1
 
$( ".selector" ).accordion({ collapsible: true });

previousOpenType: Boolean

Default: false
if true then previous section remain open while active to other section.
Code examples:

Initialize the accordion with the previousOpen option specified:

1
 
$( ".selector" ).accordion({ previousOpen: true });

openAllType: Boolean

Default: false
Whether to show all section open(active) at once. Allows collapsing to each section.
Code examples:

Initialize the accordion with the openAll option specified with collapsible:true option

1
 
$( ".selector" ).accordion({ collapsible: true, openAll: true });

iconsType: Object

Default: { "header": "acordUp", "activeHeader": "acordDown" }

Icons to use for headers. Set to false to have no icons displayed.

  • header (string, default: "acordUp")
  • activeHeader (string, default: "acordDown")
Code examples:

Initialize the accordion with the icons option specified:

1
 
$( ".selector" ).accordion({ icons: { "header": "acordUp", "activeHeader": "acordDown" } });
